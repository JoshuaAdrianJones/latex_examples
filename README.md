# LaTeX Examples

A curated collection of LaTeX templates demonstrating various features and packages for academic writing, mathematical proofs, colored presentations, and process documentation.

## Overview

This repository provides ready-to-use LaTeX templates that showcase best practices and advanced features. Each template is fully documented and can be used as a starting point for your own documents.

## Templates

### 1. Colored Boxes with tcolorbox
**Location:** `tcolorbox_examples/box.tex`

Advanced examples of the tcolorbox package featuring:
- Colored boxes with custom styling
- Two-part boxes with `\tcblower`
- Side-by-side code listings with rendered output
- Custom theorem environments with colored backgrounds
- Breakable boxes that span multiple pages
- Watermarks and nested boxes
- Fancy title boxes with independent styling

**Required packages:** `tikz`, `tcolorbox[most]`, `lipsum`, `lmodern`

**Compile:**
```bash
cd tcolorbox_examples
pdflatex box.tex
```

### 2. Academic Article with Proofs
**Location:** `example_proof_article/example_proof_article.tex`

Complete academic article template demonstrating mathematical proofs with:
- Double-spaced formatting (12pt, A4, 1-inch margins)
- Mathematical proof environments (`amsthm`, `amsmath`)
- Bibliography management
- Formal mathematical notation and equations
- Structured sections and paragraphs

**Required packages:** `geometry`, `babel`, `amsmath`, `commath`, `amsthm`, `amssymb`, `setspace`

**Compile:**
```bash
cd example_proof_article
pdflatex example_proof_article.tex
bibtex example_proof_article
pdflatex example_proof_article.tex
pdflatex example_proof_article.tex
```

### 3. Mathematical Proofs Template
**Location:** `math_proofs/proofs.tex`

Comprehensive template for mathematical documents with:
- Custom theorem environments: axiom, proof, theorem, lemma, problem, exercise, reflection, proposition, corollary
- Predefined math set commands (`\N`, `\Z`, `\Q`, `\R`, `\C`, `\I`)
- Custom headers with author and date using fancyhdr
- Table of contents and automatic list of theorems
- Number system diagram reference

**Required packages:** `graphicx`, `geometry`, `amsmath`, `amsthm`, `amssymb`, `thmtools`, `fancyhdr`

**Compile:**
```bash
cd math_proofs
pdflatex proofs.tex
```

### 4. Process Documentation
**Location:** `tex_processes/process.tex`

Template for creating operational procedures and process flows with:
- TikZ flowcharts with professional styling
- Flowchart shapes: start/stop, input/output, process, decision
- Margin notes for additional commentary
- Highlighted information boxes
- Asymmetric page layout (extra margin space for notes)
- Customizable enumeration lists

**Required packages:** `tikz` (with shapes, arrows.meta, arrows libraries), `enumitem`, `hyperref`, `tcolorbox`, `geometry`, `marginnote`

**Compile:**
```bash
cd tex_processes
pdflatex process.tex
```

## Prerequisites

You need a LaTeX distribution installed on your system:

- **macOS:** [MacTeX](https://www.tug.org/mactex/)
- **Windows:** [MiKTeX](https://miktex.org/) or [TeX Live](https://www.tug.org/texlive/)
- **Linux:** TeX Live (usually available via package manager)

## Usage

1. Clone this repository:
   ```bash
   git clone https://github.com/yourusername/latex_examples.git
   cd latex_examples
   ```

2. Choose a template and navigate to its directory

3. Compile the template:
   ```bash
   pdflatex template_name.tex
   ```

4. Modify the template for your needs:
   - Update document metadata (title, author, date)
   - Replace placeholder content
   - Adjust packages and styling as needed

## Customization Tips

### Changing Colors
For tcolorbox examples, modify color specifications:
```latex
\begin{tcolorbox}[colback=blue!5!white,colframe=blue!75!black]
```
where `blue!5!white` means 5% blue, 95% white.

### Adding Theorem Environments
In mathematical templates, create new theorem types:
```latex
\newtheorem{definition}{Definition}
\newtheorem{example}{Example}
```

### Customizing Page Layout
Adjust margins and paper size:
```latex
\usepackage[a4paper,margin=2.5cm]{geometry}
```

### TikZ Flowchart Customization
Modify node styles in process templates:
```latex
\tikzstyle{process}=[rectangle,minimum width=3cm,fill=orange!30]
```

## Generated Files

Some templates generate additional files during compilation:
- `tcolorbox_examples/box_ex.tex` - Auto-generated by the savelowerto feature
- `*.aux`, `*.log`, `*.toc` - LaTeX auxiliary files
- `*.pdf` - Compiled PDF output

These files are automatically gitignored and should not be committed.
